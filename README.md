# PROJECT-EXPENSE-CONTROL
# DESCRIPTION
-  A simple web application to register and manage daily expenses.
# PURPOSE
-  Practice Javascript and DOM manipulation while building a useful finalcial control tool.
# Features
-  Add daily expenses
-  View a list of expenses
-  Simple and clean interface
-  Runs directly in the browser
# Technologies used:
-  HTML
-  CSS
-  Javascript
# Project Structure:
-  index.html - Page structure
-  style.css - Styling
-  script.js - Application logic
# How to use:
-  Clone the repository
-  Opne index.html in your browser
-  Start adding your expenses
# Contributions:
-  Suggestions and improvements are encouraged.
